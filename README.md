## 📋 Описание

Скрипт выполняет автоматическую настройку сервера под управлением **Ubuntu/Debian**, включая:

- 🔐 Защиту SSH (ключи, смена порта, отключение паролей)
- 🛡️ Настройку фаервола UFW
- 🚀 Оптимизацию сети (TCP BBR)
- 💾 Управление Swap-файлом
- 🔄 Автоматические обновления безопасности
- 🚫 Защиту от брутфорса (Fail2ban)
- 🌐 Cloudflare WARP (VPN/Proxy)
- 🎛️ Панель управления 3x-ui

Все действия логируются в `/var/log/server_setup.log`.

---

## ✨ Возможности

| Компонент | Функция | Описание |
| :--- | :--- | :--- |
| **SSH** | Ключи + Порт | Генерация/замена ключей, смена порта (по умолчанию 2222) |
| **UFW** | Фаервол | Активация, открытие портов, правила ICMP |
| **BBR** | Сеть | Включение TCP BBR + fq для ускорения трафика |
| **Swap** | Память | Создание файла подкачки 1-3 ГБ |
| **Updates** | Обновления | `unattended-upgrades` с авто-ребут в 03:00 |
| **Fail2ban** | Защита | Бан после 3 попыток, email-уведомления |
| **WARP** | Cloudflare | Режим VPN или SOCKS-прокси (порт 40000) |
| **3x-ui** | Панель | Автоматическая установка с коррекцией cron |
